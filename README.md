# Neon Danmaku STG

ブラウザで動作する、ネオン・サイバーテイストの縦スクロール弾幕シューティングゲーム。

## 1. 概要
- **技術スタック:** HTML5, CSS3, JavaScript (ES6+), Canvas API
- **コンセプト:**
    - ネオンカラーのビジュアルと加算合成による発光演出。
    - サイバーグリッドとスターフィールドによる疾走感のある背景。
    - 調整されたオートパイロット（AI）によるデモプレイ。

## 2. ファイル構成
プロジェクトは関心の分離に基づき、以下の3ファイルに分割されています。
- `index.html`: ゲームの構造とUIレイヤー。
- `style.css`: ネオンテイストのスタイル定義。
- `game.js`: ゲームエンジン、エンティティ管理、描画ロジック。

## 3. 操作方法
- **移動:**
    - **マウス/タッチ:** ドラッグによる直感的な移動。
    - **PC:** 矢印キー（`Shift` で低速移動と当たり判定の表示）。
- **攻撃:**
    - `Z` キーまたは画面クリック（オート射撃）。
    - `X` キー（ボム/BURST）：画面内の敵弾を消去し、敵にダメージを与える。
- **オートモード:**
    - `A` キーまたはUIボタンで切り替え。AIが回避、攻撃、アイテム回収を自律的に行います。

## 4. v2 で追加・改善された機能
- **ビジュアル演出:**
    - `lighter` 合成によるネオン発光エフェクトの強化。
    - サイバーグリッド背景と流れるスターフィールドの実装。
- **ゲームシステム:**
    - **ボスの出現猶予**: スコア到達後、インターバルを経てボスが登場。
    - **ボスのHPゲージ**: 動的に色が変化するネオンHPバーを画面上部に表示。
    - **AI緊急ボム**: オートパイロット時に絶体絶命の状況で自動的にボムを発動。
- **リファクタリング:** コードを機能ごとに外部ファイルへ分離し、拡張性を向上。
